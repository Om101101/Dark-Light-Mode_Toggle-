# Dark-Light-Mode_Toggle-
A clean and optimized dark/light theme toggle built using pure JavaScript. It detects system theme preference, allows manual switching, and stores user choice using localStorage.


Create a clean, optimized, and production-ready Dark / Light Mode Theme Toggle using Vanilla JavaScript.

Requirements:
- Detect system theme using prefers-color-scheme
- Apply theme on first load
- Allow manual toggle using a button
- Save user preference in localStorage
- If user has not selected a theme, follow system theme changes automatically
- Use minimal and optimized code
- Avoid duplicate logic
- Use clean function-based approach
- Write readable variable names and comments

Deliverables:
1. Optimized JavaScript code
2. Sample HTML button
3. Basic CSS for dark and light themes
